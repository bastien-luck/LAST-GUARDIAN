LAST GUARDIAN

Plateforme utilisé :
Windows 7

Logiciel de programmation :
Qtcrearor

Ce projet a été réalisé lors de ma seconde année spécialité informatique de confiance : cyber sécurité du logiciel à l'ENSIBS.

Le jeu est plus ou moins expliqué dans le rapport.
Le jeu se joue avec la souris pour tous ce qui est de type bouton et avec le clavier pour le déplacement (flèche haut/bas), tir (espace), choix d'arme ( & é " ' ( ).
Le dernier niveau peut sembler impossible avec la mauvaise méthode mais il est bien finissable.

Pour la documentation du code : se référencer au dossier "html".
Pour lancer le programme : se référencer au dossier executable.

Le jeu n'est pas vraiment fini mais ne sera pas modifié donc il est considéré en version finale.

Ce projet a été fait en partant d'un choix personnel qui devait tenir en compte quelques contraintes d'où certaines parties du code faite différemment de la majeure partie du code.
Certains fichiers du rendu ne sont pas présent car je tiens à limiter aux sources, exécutable et rapport dans mes projets git hub.
